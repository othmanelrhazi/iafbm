Module d'interrogation

Le module d'interrogation permet d'effectuer des requêtes simulant une interrogation de type fulltext sur les modèles.

Implémentation

L'interrogation est implémentée dans les classes iaExtRestController et iaModelMysql.

Le pendant controller effectue:

  • le basculement au mode d'interrogation par la détection de la présence du paramètre xquery
  • la création des paramètres destinés au modèle, contenant les champs, comparateurs, opérateurs et valeurs pour l'interrogation

Le pendant modèle effectue:

  • la mise à disposition du template d'interrogation query (commun ou redéfini)
  • l'interrogation SQL en utilisation le système de templating

Processus d'interrogation

Lorsque le paramètre xquery est présent dans une requête, le processus suivant est effectué:

  • construction des paramètres d'interrogation
  • application du template d'interrogation query
  • exécution de la requête SQL d'interrogation
  • envoi des résultats
