Structures de données ExtJS

L'application fait usage de définitions factorisées pour le contenu réutilisable des grilles et formulaires.

Implémentation

Grilles

Les grilles factorisées prennent le préfix iafbm.grid.common et sont implémentées dans le fichier grids.js. Ces dernières sont des extensions de Ext.ia.grid.EditPanel, le widget de grilles ExtJS spécifique au projet.

Formulaires

Les formulaires factorisés prennent le préfix iafbm.form et sont implémentées dans le fichier forms.js. Ces derniers sont des extension de Ext.ia.form.Panel, le widget de formulaires ExtJS spécifique au projet.
