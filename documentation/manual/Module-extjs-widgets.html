Widgets ExtJS

D'une manière générale, les extensions des widgets ExtJS sont documentés dans la partie documentation API.

Implémentation

Les widgets se décomposent en plusieurs catégories

  • Types de colonnes et champs
  • Grilles et formulaires
  • Popups et notifications
  • Widgets standalone
  • Widgets propres aux commissions

Types de colonnes et champs

Plusieurs types de colonnes et de champs afin de standardiser et adapter le comportement et le visuel aux besoins du projet. Voici les éléments remarquables:

  • Ext.ia.grid.column.ActionForm
    permet de charger les données d'une ligne de grille au sein du formulaire correspondant et de l'afficher dans une fenêtre de type popup (utilise Ext.ia.window.Popup)
  • Ext.ia.grid.RadioColumn
    simule un champs de grille de type radio, inexistant dans le framework ExtJs
  • Ext.ia.form.field.Percentage
    permettent de formatter les données de type pourcentage au sein d'une colonne de grille ou d'un champs de formulaire
  • Ext.ia.form.SearchField
    permet d'afficher un champs de recherche type fulltext avec gestion automatique du paramètre d'interrogation xquery

Grilles et formulaires

  • Ext.ia.grid.EditBasePanel et Ext.ia.grid.EditPanel
    est un grille éditable prête à l'emploi fournissant les fonctionnalités suivantes:
    • boutons d'ajout/suppression de lignes (enregistrements) et enregistrement des modifications
    • champs de recherche de type fulltext
    • paging
    • basculement en mode éditable/non-éditable du formulaire
  • Ext.ia.form.Panel
    fournit les fonctionnalités suivantes:
    • chargement de données à partir d'un nom de modèle et une valeur de clé primaire ou à partir d'une instance de modèle ExtJS
    • affichage d'une version antérieur d'un enregistrement (versioning)
    • détecter globale des modifications non-enregistrées
    • basculement en mode éditable/non-éditable du formulaire

Popups et notifications

  • Ext.ia.window.Popup
    fournit une fenêtre modale de dimension standard permettant de contenir un composant ExtJS arbitraire, avec système de notification de modifications non-enregistrées pour les composants de type form
  • Ext.ia.window.Notification
    fournit un système de notification standard

Widgets standalone

  • Ext.ia.Versioning
    fournit un widget pour le versioning, permettant
    • la visualisation des versions
    • la sélection d'une version
    • l'ajout de versions (*type tag*)

Widgets propres aux commissions

  • Ext.ia.tab.CommissionPanel
    est une extension du widget tab ExtJS fournissant les fonctionnalités suivantes:
    • indicateur d'état de phase (terminé) sous forme de pastille
    • indicateur de commission clôturée sous forme de message utilisateur
    • notification de modifications non-enregistrées
  • Ext.ia.form.CommissionPhasePanel
    est une extension du widget formulaire spécifique au projet fournissant les fonctionnalités suivantes:
    • case à cocher pour l'indicateur d'état de phase (terminé)
