Process

  • Testing
    • UnitTests should run at least once a day on the development branch
    • The developer should run UnitTests before every commit
  • Writing
    • The developer should write UnitTests for every PHP feature he develops

    • Versioned entities retrieval
    • Versioned entities retrieval joins
    • Versioned entities retrieval with n..n relations

  2. Entities commons

    • Search

  3. Catalogs

    • Read-only check

  4. Commissions

    • Closed commission write lock

  5. Processes

    • Closed commission, data freeze check
      • Add personne to commission; close commission; modify personne; check that personne in commission is not modified
        • Personne modifications to be tested
          • Nom, prénom, genre, pays

          • Formations
          • Carrière professionnelle
          • Deletion
