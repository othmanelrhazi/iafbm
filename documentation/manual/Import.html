Import de données externes

L'application utilise des données UNIL et CHUV fournies au format CSV. Un script d'import permet d'intégrer ces informations dans la base de données iafbm.

Ces données sont conservées par Stefan Meier.

Utilisation

  1. Placer les fichiers CSV d'import dans le répertoire iafbm/import/
  2. Lancer le script d'import 
    cd iafbm/scripts
php import.php

Notez que cette étape est effectuée automatiquement par le script de mise-à-jour de l'application, cf recette de déploiement.
