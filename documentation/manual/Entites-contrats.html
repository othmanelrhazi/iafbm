Terminologie

  • Section: SSF ou SSC
  • Activité Type: Type de contrat (Fonction administrative SSF, fonction hospitalière SSC, ...)
  • Activité Nom: PO, PAS, PAST... Fonction administrative 1, fonction administrative 2... etc
  • Activité: Activité d'un type donnée, exercée au sein d'une section donnée

Structure organisationnelle

  • Types d'activité:

    • SSC: Fonction administrative, fonction académique, autre mandat.
    • SSF: Fonction administrative, fonction académique, autre mandat, fonction hospitalière, titre académique.

  • Listes communes

    • Activités du type Fonction académique: commune entre SSF et SSC.

Structure relationnelle

Cette structure permet d'allouer une activité (p.ex. PO) à un type (p.ex. fonction académique) et une section (p.ex. SSC).

                                   +---------+
+-------------------+    +---------| section |
| Activité          |    |         +---------+
+-------------------+    |
| id                |    |         +---------------+
| section_id        |----+     +---| activite_type |
| activite_type_id  |----------+   +---------------+
| activite_nom_id   |-------+
+-------------------+       |      +--------------+
                            +------| activite_nom |
                                   +--------------+

Réflexion (archive)

Relations section, type, activité

Alternative 0

+---------+        +------+        +----------+
| Section |--------| Type |--------| Activite |
+---------+  1..1  +------+  1..n  +----------+
  • Abandonnée: Equivalences des Types impossible
  • Equivalences Type entre les Sections (p.ex. Mandat)
  • Doublons Activité (p.ex. PO)

Alternative 1

+---------+        +------+        +----------+
| Section |--------| Type |--------| Activite |
+---------+  1..n  +------+  1..n  +----------+
  • Doublons Type (p.ex. Mandat)
  • Doublons Activité (p.ex. PO)

Alternative 2

+---------+        +------+        +----------+
| Section |--------| Type |--------| Activite |
+---------+  1..n  +------+  n..n  +----------+
  • Doublons Type (p.ex. Mandat)
  • Activité commune à plusieurs Types (p.ex. PO)

Alternative 3

+---------+        +------+        +----------+
| Section |--------| Type |--------| Activite |
+---------+  n..n  +------+  n..n  +----------+
  • Type commun à plusieurs Activités (p.ex. Mandat)
  • Activité commune à plusieurs Types (p.ex. PO)
