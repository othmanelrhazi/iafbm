Concept archivage

Aspect technique

Stockage

Lors de l'archivage, le tuple de la table ainsi que tous les tuples relatifs (jointures) sont stockés dans une table archive. Chaque colonne des tuples à archiver sont stockés sous forme de ligne dans la table archive dont la structure est la suivante:

  • nom de la table
  • nom du champs de la clé primaire
  • valeur du champs de la clé primaire
  • nom du champs (colonne)
  • valeur du champs (colonne)

Récupération

Les données stockées par l'archivage permettent d'implémenter aisément un algorythme simple qui re-crée les tuples archivés dans une base de données vierge.
